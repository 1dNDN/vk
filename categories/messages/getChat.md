# Метод Messages.GetChat
Возвращает информацию о беседе.

## Синтаксис
```csharp
public Chat GetChat(long chatId)
```

## Параметры
+ **chatId** - Идентификатор беседы.

## Результат
После успешного выполнения возвращает список объектов, описывающих беседу (мультидиалог).
